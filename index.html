Sub-downloader

A way to download Subtitles faster than you think. Works on OS X.

Download Subtitles in less than 3 clicks

Sub-Downloader is a Mac OS X service by means of which you can download subtitles just by right clicking on a video file and going to services and then Get Subtitles. Subtitles will be downloaded subject to their availabity on thesubdb, since the script uses its API. Final result is something like:

Installation

Just download the zip file, unzip it, place the so created folder in your home directory. Then run this is terminal:
What this script does is simply install requests for Python. If you have previously installed it, skip running the script. Finally create a service in automator. Head over to the wiki and follow the instructions. It will only take a few minutes!!

A look at the scripts

The scripts used for downloading the subtitles are written in Python. The script that is called by the service is main_script.py. What this script does is take a path to a movie file as an argument and just download the subtitles for that movie in the same directory in which it is located. meta_extract.py simply extracts file name, extension and directory of given file from its path. hash_video simply calls a hash function provided by thesubdb whose API the script uses.
