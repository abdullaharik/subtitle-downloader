Sub-Downloader

A way to download subtitles faster than you think (Currently for OS X only)

Installation

  • If you have git installed
    • head over to GitHub and follow the README.
  • Else
    • Download the ZIP file from GitHub, and extract it.
    • Rename the so-created folder to Sub-Downloader
    • Move it to your Home Directory.
    • Move the Get Subtitles folder. Type the code in the next point in terminal.
    • mv ~/Sub-Downloader/Get\ Subtitles ~/Library/Services/
