Sub-downloader

A way to download Subtitles faster than you think. Works on OS X.
download .ZIPdownload .TGZ

Download Subtitles in less than 3 clicks

Sub-Downloader is a Mac OS X service by means of which you can download subtitles just by right clicking on a video file and going to services and then Get Subtitles. Subtitles will be downloaded subject to their availabity on thesubdb, since the script uses its API. Final result is something like:

Installation

Just download the zip file, unzip it, place the so created folder in your home directory. Then run this is terminal:
cd ~/Sub-Downloader
chmod +x install.sh
./install.sh
 Finally create a service in automator
